Slackware 简介

Slackware 发行版不是我使用的第一个 GNU/Linux。我第一次登陆类 Unix 操作系统是在 TurboLinux 下，接着是 RedHat。然而我却感觉 Slackware 带给我的最多……

Slackware 是由 Patrick 创建和维护的发行版，至今已发行超过 10 年。它遵循 KISS 的原则，给了用户很大的自由度，因此，Slackware 也是最 Unix like 的 Linux 发行版。这里将告诉你为什么选择 Slackware，当然，这也只是一家之言。

开心玩转 Slackware

这里是一些我收集的文章和一点点的翻译。选择 Linux 作为日常的工作系统， 其实是很快乐的，:)

alias fun='Slackware Linux'

alias fun='Slackware Linux' 是一篇翻译自 The Slack World 第一期（4 月号）使用系统小窍门栏目里的文章，介绍了如何使用别名，使得面对系统的日常生活更简便些。

别名非常的有用，通过简单的组合就能使用操作系统提供的功能来控制机器，来完成本来很麻烦的事情。比如打开关闭光驱，在文本界面下产生特定目录下的 mp3 列表来播放歌曲等等。在没有学会别名之前，完成这些工作需要通过一系列确定的步骤，但是通过建立自己的别名，可以将几步结合在一起，省了很多力气，太酷了～

软件推荐

Emacs

Emacs 是一个拥有几乎无限可扩展能力的文本编辑器，如果你稍微懂一点 elisp，将更能体会这种感觉。

Opera 浏览器

Opera 浏览器除了在教科书上标称的速度最快，我觉得它和其他浏览器很大的区别在于近乎完全的可定制性，很人性化；它包含巨丰富且实用的功能，在一个小小的浏览器内能满足你很大一部分冲浪需求……

Mutt + esmpt + fetchmail

Mutt + esmpt + fetchmail 这篇文章得感谢 AMD-K6 的用 MUTT 收发邮件这篇帖子，给我们这些邮件不多的兄弟方便的同时，有了免开 sendmail、postfix 这些著名 daemon 的机会……
