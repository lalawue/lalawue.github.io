MAXIMA in Windows

用 MAXIMA 来做微分题最好不过了，而且还可以用 EMACS 来作前端，相当地方便， 这在 Linux 下很快就可以搞定，但是在 Windows 下却费了一番功夫。

首先当然是要下载 MAXIMA for Windows 的版本安装了，然后到 MAXIMA 的安装目 录寻找 maxima.bat，大概的相对路径为 ./lib/maxima/5.9.0/binary-gcl/（比如我装 的是 5.9.0），将这个绝对路径放到你的系统或是个人 %PATH% 变量里面，这样就 可以在 cmd 下启动了。

然后呢和 Linux 下一样，将设置 EMACS 的 .el 文件放到 load-path 里面 （maxima.el 里有说明），设置好你的 .emacs 后，加上一句话 (setq maxima-command "maxima.bat") 就可以了。

P.S: maxima.bat 里只有一句话 maxima.exe -eval "(user::run)"，大概 maxima.exe 并不是交互运行的。

Slackware 11 is release !

昨天中午用 slapt-get 将系统跟上了 current 的脚步，刚升级完安装包，就发现 root 有了新邮件，这大概就是新版本发布的信号了，每次升级到一个新的版本， root 都会有欢迎来到 Linux 的邮件的。

果然没错，这回终于到 11 了，不过系统的变化没有期待中的那么大，内核还是 2.4 系列的，已经到了 2.4.33，本来还以为可以转到 2.6 的。

其他的变动就都很平常了，现在应该也提供 ISO 下载了吧。

越狱成功

今天中午 11:20，松鼠越狱成功，从小笼子里冲了出来，先是窜进了屋子，待我们 欲捉它时，又成功突围，进入了阳台。

正当我们洋洋得意地关上阳台门，只留一个人在阳台上逮它，以为必定成功的时候。 松鼠从阳台内的污水管道借着阳台内粗糙的小碎石贴面爬了上去，于是我们用撑衣 杆赶它下来。可是聪明的松鼠荡到了凉晒的衣服上面，勇敢的从三楼跳下了院子， 成功突围，然后渐渐地逃离我们的视线。

越狱三次，终于成功了……
