2005-01-30, 星期日

XP update....

前几天刚刚搞好了 slk 的安装和配置，家人说 xp 现在有点慢了，我又搞不出什么名堂，于是就 update 吧，让机器里的虫虫不起作用。本来以为避免安装那些 sp1、sp2 就可以了，后来才发现我错了……

Update 完毕之后重启，结果没有象以前一样出现 30 天试用的警告信息，倒是出现了输入激活码才能进入的提示。靠，连 xp 都进不去，混个屁。幸好还有个 Linux，换个系统查了半天，google 得到的多是算号器，我晕……最后我放弃了，第二天买了张 D 版搞了一个中午。以后打死我都不 update 了。

幸好我没买正版，要不然每次 update 之后都要重启然后输密码，花这么多钱瞎折腾，我不气死才怪。

主页的标题、字体……

刚设计这个网页的时候，我用的是 Luxi Serif，但是这是 Linux 下独有的（？），反正 IE 上显示很差，然后自己加上了 sans-serif，然后在 IE 下可以了，但总感觉不是很爽。试过 comic sans ms，在 IE、Opera 里面真是爽，但是在 Linux 下，Firefox 和 Netscape 都无法正常显示，想不清楚什么地方出了问题~

“Journal” 似乎不是很好用

EmacsWiki 里的 Journal 似乎不是很好用，不是很爽的样子，或许也是我不懂用吧，难搞……

最后还是决定自己来处理这些麻烦的类似 blog 的日记，其实也蛮容易的，靠着emacs，很多重复的连接都只要复制即可。再说，大不了就多做点体力劳动，哈哈。

准备处理自己原来在 donews blog 上的一些文章，虽然写得很渣，内容少，但是毕竟记录了自己的一段日子，从开始使用 slackware，到现在慢慢熟悉、了解GNU/Linux，还有其他半个学期来发生的值得回忆的东东……这段时间过得真快，而且进步也很快，总有时间不够用的感觉。

记得刚开学的时候，我还不了解 blog，接着申请了一个，记录自己学习、进步的点点滴滴……可现在，自己的主页都出来了。

2005-01-03, 星期一

用 IE 浏览出现问题

昨天刚刚考了两门课，物理和期末英语。上午一门下午一门，累死了。因此也没有什么心思来照看看自己的地盘。今天突然心血来潮，觉得自己的网页在 Opera 里看着蛮爽，但是面对广大群众的 IE 会怎么样呢，如果表现很差，这个网页就几乎没有存在的必要了。

不试不知道，一试吓一跳。在同学的 IE 上面显示差别蛮大的。自然，我也顺便在自己的机子上试了一下 Mozilla、Firefox 以及 Netscape 之类的（我的浏览器似乎有点多了~~），差别是有，但不如 IE 的大。

因为网页结构都是一样的，差别就在各个浏览器对 css 的语法解释上面。郁闷了一阵子，最后修改了一下 css，终于搞定。对于自己的网页，主要还在于padding 和 margin 这两个关键字和 table 的配合。

我对 css 还不是很了解，也不想深究，既然最后我的网页对各路浏览器差别不大，我也就罢休了。

呵呵，休息一下~
